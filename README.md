처음부터 사용법

로컬 PC 실행:
1. install_requirements.bat 더블클릭
2. create_desktop_shortcut.bat 더블클릭
3. 바탕화면의 MARU KRA FINAL AI 아이콘 더블클릭

클라우드 실행:
1. GitHub에 폴더 업로드
2. Streamlit Community Cloud에서 app.py 배포
3. 생성된 주소를 핸드폰 크롬에서 열기
4. 홈 화면에 추가

대시보드 사용:
1. 실시간 수집/재분석 클릭
2. 추천 1위와 TOP3 확인
3. 변경감지 탭에서 기수/마번/줄/거리 변경 확인
4. 한달우승원인 탭 확인
5. 공식 구매/확인 페이지 열기
6. 직접 확인 후 수동구매 기록
