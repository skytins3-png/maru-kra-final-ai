# MARU KRA GREEN FOCUS STRONG RECOMMEND ENGINE

현장용 녹색 집중화면 버전입니다.

## 수정 내용
- 최종결과를 녹색 큰 박스 안에 집중 표시
- 오늘조합 강력추천 박스 추가
- 신뢰도 / 예상배당 / 추천금액 / 판정만 크게 표시
- API 표, 시뮬레이션, 로그, URL 예시는 숨김처리
- 필요한 경우 펼쳐보기로 확인
- 관망이면 조합 '-' 처리
- 강력추천 조건 만족 시 크게 표시
- 허브 저장 / Telegram 알림 기능 유지
- 경주시간표 30분 전 자동분석 유지
- API 19개 실시간 수집 유지
