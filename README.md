# MARU KRA FINAL CLEAN 19 API DASHBOARD

깨끗하게 새로 만든 최종형 Streamlit 대시보드입니다.

## 특징
- PC 이미지형 대시보드 UI
- API 19개 입력 지원
- API Key 숨김 처리
- URL 예시에서 serviceKey 숨김
- 보조 API 9~19번은 접기/펼치기 처리
- 한국시간 KST 기준
- chulNo 기준 1~14번만 추천
- enNo/hrNo/chaksun/age 숫자 추천 차단
- 최종결과 크게 표시
- 관망/추천금액 0원 방어
- 과거 추천 로그 저장

## GitHub 업로드
압축 해제 후 아래 2개만 업로드하세요.

- app.py
- requirements.txt

## 사용
1. Streamlit Cloud에 app.py / requirements.txt 업로드
2. 왼쪽 사이드바에서 API Key와 19개 URL 입력
3. 설정 저장
4. 데이터 불러오기
5. 최종결과 확인

## 주의
자동구매 기능이 아니며, 수익을 보장하지 않습니다.
