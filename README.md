# MARU KRA FULL 19API REALTIME ENGINE

형님이 어렵게 만든 1~19번 API 구조를 다시 살린 전체 실시간판입니다.

## 핵심

- 1~19번 API 전부 입력/호출
- API Key 입력 유지
- 한국시간 기준
- 서울/부산경남/제주 + 경주번호 선택
- 실시간 데이터 불러오기
- 자동 새로고침 선택
- chulNo 기준 마번 분석
- 1~14번 범위만 추천/시뮬레이션
- 배당/체중/장구/인기/코너/기록/기수변경 등 연결된 API는 모두 점수에 반영
- API Key 숨김 URL 예시
- 오류 API가 있어도 정상 API만으로 분석 계속
- 관망 안전장치
- 자동구매 아님, 수동 판단용

## 중요

API가 실제로 현재 경주 데이터를 반환하면 앱은 즉시 분석에 반영합니다.
다만 공공데이터 API 자체가 비어 있거나 HTTP 500이면 해당 API는 제외하고 분석합니다.
