MARU KRA 통합본 - 18,000원 삼쌍승 18장 / 허브 분석 / 모바일 빠른구매 대시보드

핵심 반영:
- 19개 API URL 내장, URL 재입력 없음
- API Key는 Streamlit Secrets / GitHub Actions Secrets에서 자동 읽기
- API ON/OFF 유지
- 경주일정표/출전마/말정보/과거성적은 아침 1회 캐시
- 체중/기수변경/주로/날씨는 경주 60~20분 전 갱신
- 배당/인기/예측계열은 경주 20분 전부터 5분마다 갱신
- 모든 API를 매시간 19개 전부 호출하지 않음
- 허브가 저장 데이터 + 최신 데이터 합쳐 분석
- 강력추천 경기만 모바일에 표시
- 추천창 3개: 안정형 / 변수형 / 고배당형
- 각 추천창 3마리 조합을 6순열로 전개, 총 18장
- 18장 × 1,000원 = 18,000원 기준
- 과거 부진마가 기수변경, 체중변화, 경기장/거리/주로/배당 변수로 급상승하는 케이스를 변수점수/고배당점수에 반영
- 모바일은 mobile_recommend.json 1건 우선 읽기로 메모리/속도 개선
- 모바일 빠른구매 모드 추가:
  · 10초 급함 = 1조합 6장 / 6,000원
  · 30초 가능 = 1조합+2조합 12장 / 12,000원
  · 60초 가능 = 전체 18장 / 18,000원
- 모바일 복사/붙여넣기 보조:
  · 전체 18장 복사용 텍스트 표시
  · 1조합/2조합/3조합별 6장 복사용 텍스트 표시
  · 전체/조합별 텍스트 파일 다운로드
  · 공식 마권구매 페이지 열기
- PC는 기존 전체 분석/관리/통계 유지
- 자동구매/자동결제 없음. 공식 구매 페이지에서 사용자가 직접 입력/확정

실행:
streamlit run app.py
