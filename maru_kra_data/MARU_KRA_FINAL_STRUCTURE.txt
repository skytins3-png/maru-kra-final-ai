MARU KRA 최종 통합 구조

목표
- 모바일: 가볍게 보기 / 조합 복사 / 공식 구매페이지 이동
- 허브: KRA API 및 공식 메뉴 데이터 수집 / 분석 / 저장 / 모바일 추천 JSON 생성
- PC: 전체 확인 / 관리 / 오류점검 / 빅데이터 검토

1. 모바일 역할
- 추천 요약 화면
- 경주시간 / 추천저장 / 상태 표시
- 안정형 / 변수형 / 고배당형 3추천창
- 10초 수동구매 모드
- 1조합 6장 / 2조합 6장 / 3조합 6장 빠른 복사
- 공식 마권구매 열기
- 모바일 허브분석 실행
- 결과 / 환급 / 손익 간단 표시
- 모바일은 무거운 전체 표, API 상태표, 빅데이터 표, PDF 원문을 기본 표시하지 않음

2. 허브 역할
- 경마캘린더 수집/저장
- 경주시간표 수집/저장
- 경주개요 수집/저장
- 경주안내/출전마/기수/말/조교사 정보 저장
- 금일출전마체중 저장
- 경주속보/출전취소/기수변경 저장
- 예상배당/인기흐름 저장
- 주로/날씨/기상특보 반영
- 사전분석 실행
- 경주 20분 전 실시간 보정
- mobile_recommend.json 생성
- 경주 후 결과/확정배당/적중/환급/손익 저장
- 빅데이터 로그 누적

3. PC 역할
- 전체 분석 대시보드 확인
- 26개 API 상태 확인
- KRA 메뉴 수집 상태 확인
- 경주시간표 전체 확인
- 출전마/기수/조교사/말 점수 확인
- 마체중 변화표 확인
- 예상배당 변화 확인
- 추천 근거 확인
- 모바일 추천 미리보기
- 결과/손익/빅데이터 확인
- 오류 로그 확인
- API ON/OFF 관리

4. 데이터 흐름
KRA API / KRA 공식메뉴
→ 허브 수집
→ 허브 분석
→ mobile_recommend.json 생성
→ 모바일 1번 추천 화면
→ 모바일 2번 빠른구매 화면
→ 공식 구매페이지 이동
→ 경주 후 결과 저장
→ PC에서 확인 및 빅데이터 누적

5. 운영 시간표
- 월~목: 대기모드 / 캘린더만 확인 / 지난 결과 정리
- 금토일 아침: 경마캘린더 + 오늘 경주시간표 + 출전마/기수/말 사전분석
- 각 경주 60분 전: 출전정보, 취소, 기수변경 재확인
- 각 경주 20분 전: 체중, 배당, 속보, 주로, 날씨 반영 후 최종 추천 생성
- 경주 후: 결과, 확정배당, 적중/실패, 환급/손익 저장

6. 자동구매 제한
- 이 앱은 자동구매/자동결제 기능을 포함하지 않음
- 공식 구매페이지로 이동 후 사용자가 직접 입력/확정
